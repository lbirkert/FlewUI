# FlewUI

A Svelte 5 component library with design tokens, form utilities, and pre-built components.

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

Output goes to `dist/` for use as a package dependency.

## Project Structure

```
flewui/            # Component library source
examples/
  lightdrive/      # Example app: self-hosted cloud storage
```

## Examples

[LightDrive](examples/lightdrive/) — A self-hosted file sharing and cloud storage application built with SvelteKit and FlewUI components.
